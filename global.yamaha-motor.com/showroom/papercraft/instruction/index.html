How to download the papercraft pattern (pdf).

How to downroad the papercraft pattern (pdf).




You must have Adobe Reader installed in order to view the Paper Craft Parts Data and the Assembly Manual.

Get Adobe Reader

1.

When the PDF text appears in the window of your Web browser.
Select "Save as" from the "File" menu.

2.

When downloading without opening the file.

[Windows]
Do a right-side click on the GET icon and then select "Save this Link as..." from the pop-up menu.

[Macintosh]
Click and hold the mouse on the GET icon and select "Save this Link as..." from the pop-up menu.

Each PDF file may not be correctly displayed and printed when your Acrobat Reader version number is not the same as the version number of each PDF files being released.
Since using Acrobat Reader Ver.5.0 seems to be very stable, please download it from Adobe's website and try again.
